Why Shipy?

  • Zero build chain. Server-rendered HTML + HTMX sprinkles.
  • SQLite first. Raw SQL until you earn an ORM.
  • Auth, sessions, CSRF, forms—batteries included.
  • Deploy on a $12 VPS with systemd + Nginx.

What you’ll ship in 30 minutes

Signup/Login → CRUD → HTMX toggle → Deploy.

Philosophy

  • Default to HTML + forms + cookies + sessions + CSRF. Earn your JS.
  • One box, one repo, one brain. Earn complexity.
  • Freedom is the KPI: time, location, mind.

Quickstart

# Create app
python -m venv .venv && source .venv/bin/activate
pip install shipy-web
shipy new demo && cd demo
shipy db init
shipy dev  # http://localhost:8000
